rOCCI-server - A Ruby OCCI Server

The rOCCI-server

Requirements

The following setup is required

  • Ruby >= 2.0
  • Bundler gem installed (use gem install bundler)
  • OpenNebula >= 4.4 if the OpenNebula backend is used
  • AWS EC2 account if the EC2 backend is used

Installation
